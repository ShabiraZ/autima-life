# 🌱 AutiMa.Life — Every Step, Together

> AutiMa = Autism + Ma (mother in Hindi) + Life
> An AI-powered assistant helping parents navigate autism daily life —
> understand triggers, log behavior episodes, and find calm guidance.

## The Problem

Parents of autistic children face meltdowns without understanding why.
They ask themselves every time:
- What did I do wrong?
- Was he hungry? Overtired?
- Did I miss a trigger?

No simple tool exists to help parents track, detect patterns,
and get real guidance in the moment.

## The Solution

AutiMa.Life is an AI assistant that:
- Listens to what happened in plain language
- Asks smart follow-up questions (hunger, sleep, therapy, environment)
- Identifies likely triggers
- Gives calming strategies immediately
- Builds a pattern picture over time

## Why I Built This

I am the parent of an autistic child. This problem is personal.
The name AutiMa reflects who this is built for —
the mothers (and fathers) living autism life every day.

Idea validated through direct conversation with autism parents in Malaysia.

## Live App

🌐 [autima-life.vercel.app](https://autima-life.vercel.app)

## Roadmap

✅ Phase 1 — AI chat prototype with trigger detection  
✅ Phase 2 — Secure backend via Vercel (no API key needed for users)  
⬜ Phase 3 — Behavior logging + session memory (Firebase)  
⬜ Phase 4 — Trigger pattern detection across sessions  
⬜ Phase 5 — PWA mobile install  
⬜ Phase 6 — Specialist/therapist connect  

## Tech Stack

- HTML / CSS / JavaScript (vanilla — no framework)
- OpenAI GPT-4o-mini API
- Vercel serverless function (secure API proxy)
- Hosted on Vercel

## How To Run Locally

1. Download `index.html`
2. Open in any browser
3. Click Start Session
4. Start chatting — no API key needed
